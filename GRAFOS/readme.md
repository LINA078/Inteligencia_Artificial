# Inteligencia_Artificial
Este codigo tiene como objetivo mostrar de manera visual cómo funcionan los algoritmos de búsqueda en grafos BFS (Breadth First Search) y DFS (Depth First Search).
El programa crea dos grafos diferentes y aplica cada algoritmo sobre uno de ellos. Durante la ejecución se puede observar paso a paso cómo el algoritmo va recorriendo los nodos del grafo. Cada paso se muestra como una animación para entender mejor el proceso de exploración.
Al final del programa se muestran los dos grafos con su recorrido completo, lo que permite comparar cómo se comporta cada algoritmo.
Este codigo fue desarrollado como parte de un ejercicio académico para comprender mejor el funcionamiento de los algoritmos de búsqueda en grafos

Para desarrollar este codigo utilise las siguientes herramientas:
Python
NetworkX para crear y manejar los grafos
Matplotlib para la visualización gráfica
Google Colab para ejecutar el código en línea

Qhe hace el programa
El programa realiza lo siguiente:
Crea dos grafos con diferentes estructuras.
Ejecuta el algoritmo BFS en el primer grafo.
Ejecuta el algoritmo DFS en el segundo grafo.
Muestra paso a paso cómo se van visitando los nodos.
Resalta visualmente el recorrido del algoritmo.
Al final muestra ambos grafos con su recorrido completo para poder compararlos

Significado de los colores 
Durante la ejecución del programa los nodos y conexiones cambian de color para facilitar la comprensión del recorrido:
Rosado: nodo actual que el algoritmo está visitando.
Verde claro: nodos que ya fueron visitados.
Gris: nodos que aún no han sido explorados.
Rojo: conexiones que el algoritmo ha recorrido
