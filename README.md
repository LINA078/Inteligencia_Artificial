ACTIVIDADES DE INTELIGENCIA ARTIFICIAL
Este repositorio contiene las diferentes actividades, ejercicon y practicas realizadas durante la materia de Inteligencia Artificial.
el objetivo de este repositorio es organizar y almacenar los trabajos desarrollados durante la materia, los cuales incluyen algoritmos de busqueda, ejemplos practicos relacionados con la inteligencia artificial.
Aqui se pueden encontrar implementaciones en codigo, explicaciones y algunos ejemplos que ayudan a comprender mejor los conceptos vistos en clase.
